# Security policy

## Reporting a vulnerability

If you find a security issue, please email [hello@praks.me](mailto:hello@praks.me) instead of opening a public issue. Include:

- A description of the issue
- Steps to reproduce
- The impact, as you understand it

I aim to respond within a few days. Fixes for confirmed issues ship as quickly as I can manage — this is a personal project, so "quickly" may mean days rather than hours.

## Scope

The site itself: `wis.praks.me` and the code in this repository.

Out of scope: vulnerabilities in third-party services this site links to (Arbeitnow, Remotive, Wikipedia, etc.) — report those to the respective maintainers.

## Thanks

I appreciate responsible disclosure. Once a fix is merged, I'm happy to credit reporters in the release notes or commit if they'd like that (just mention it in your email).
