# STEM•SPARK

A platform supporting women in STEM through structured learning, mentorship connections, and community collaboration.

## About

STEM•SPARK provides practical resources for women pursuing careers in science, technology, engineering, and mathematics. The platform focuses on creating meaningful tools that address real challenges in STEM career development.

## Features

### Core Functionality
- **Learning Paths**: Structured courses covering programming, data science, engineering, and research methods
- **Mentorship Platform**: Connect with experienced professionals for career guidance
- **Resource Library**: Curated articles, tutorials, and learning materials
- **Community Forum**: Discussions, questions, and peer support
- **Progress Tracking**: Monitor course completion and learning milestones
- **Career Resources**: Information about STEM career paths and opportunities

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom component library
- **State Management**: React Hooks

## Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/itsmepraks/womeninstem.git
cd womeninstem

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
